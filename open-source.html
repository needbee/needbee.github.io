Open-Source Software

Here are all of my GitHub projects, organized by language.

Objective-C/Cocoa

  • NBEllipsisView: dots to indicate multiple items to scroll through horizontally
  • NBGallery: enhanced UIScrollView with utility methods for paging
  • NBHighlightButton: allows changing the background color on touch
  • NBMiniModal: a small modal that doesn't cover the whole screen
  • NBThemeConfig: configure the colors for your app in a single plist
  • NBVariableTargetAreaButton: can detect taps outside of itself
  • NSArray+Contains: searches an array for multiple items at once
  • NSString+SmartTrim: trims to a given length, including prefixes and suffixes
  • UIView+PartialRoundedCorner: allows rounding of just two corners
  • UIColor+Mix: calculates colors partway in between other colors
  • UIViewController+JSBridge: easy way to communicate with JS

Ruby on Rails

  • cached_remote_record: access and cache remote records, such as web services
  • simple_auth: a simple user model with authentication features

JS/CSS

  • polyglot.me: consistent cheat sheets for multiple programming languages
  • sass-requirejs-template: template project to build Sass and Require.js using Grunt

Algorithms

  • tablesync: field-level table synchronization
